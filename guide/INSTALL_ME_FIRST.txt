================================================================================
TROUBLESHOOTING
================================================================================

Problem: "Database connection failed"
Solution: 
  - Make sure MySQL is running in XAMPP/WAMP
  - Check includes/db-connection.php has correct password
  - Database name must be: barangay_rosario

Problem: "Invalid credentials" when logging in
Solution:
  - Make sure you ran Step 4 (hash_passwords.php)
  - Try using: password (if you used Option B)
  - Check email is spelled correctly
  - Passwords are case-sensitive!

Problem: "Cannot find hash_passwords.php"
Solution:
  - Make sure you extracted the full folder
  - File is in: barangay_rosario/hash_passwords.php
  - Use Option B from Step 4 instead

Problem: CSS not loading (page looks plain)
Solution:
  - Clear browser cache (Ctrl + F5)
  - Check folder name is exactly: barangay_rosario
  - Make sure Apache is running

Problem: "Call to undefined function password_hash()"
Solution:
  - Your PHP version is too old (need PHP 5.5+)
  - Update XAMPP/WAMP to latest version
